## Welcome to the project roadmap!

If you haven't yet visited the [README](https://github.com/samanthaburton/whose_data/blob/master/README.md) for an overview, I'd suggest starting there.

The milestones below map out the first phase of the Whose Data? Project. This is a great place to explore what issues that you can contribute to. Check out the different milestones listed below, and feel free to explore the issues by label too.

Before you get started, please also check out the contribution guidelines (coming soon) and code of conduct (coming soon).

### Milestone: [Start-Up](https://github.com/samanthaburton/whose_data/milestone/1)
> Get all the basics of the Whose Data? project on GitHub, so I can invite contributors and start on research

### Milestone: [MVP Research Sprint](https://github.com/samanthaburton/whose_data/milestone/2)
> Complete enough example research to create a prototype webpage/user interface

### Milestone: [(UGLIEST) Prototype 0.1](https://github.com/samanthaburton/whose_data/milestone/3)
> Get the first prototype out in the world!

### Milestone: [Research Sprint II](https://github.com/samanthaburton/whose_data/milestone/4)
> Stretch goal: double the # of sources researched

### Milestone: [(UGLIER) Prototype 0.2](https://github.com/samanthaburton/whose_data/milestone/5)
> Iterate, iterate, iterate

### Milestone: [Mozilla Global Sprint 2018](https://github.com/samanthaburton/whose_data/milestone/6)
> Participate in the Mozilla Global Sprint

### Milestone: [Where to?](https://github.com/samanthaburton/whose_data/milestone/7)
> Evaluate & decide what's next
